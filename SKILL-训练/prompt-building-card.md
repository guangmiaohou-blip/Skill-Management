# 提示词搭建卡片

## 通用公式

```text
主体 + 关键细节 + 动作/姿势 + 构图视角 + 光线 + 背景 + 材质 + 色彩 + 风格 + 镜头语言 + 后期质感
```

## 模特图公式

```text
一张{风格}的{模特图类型}，{模特设定}，穿着/展示{服装或产品}，{姿势动作}，{构图视角}，{场景背景}，{光线描述}，{材质和颜色细节}，{镜头语言}，{后期质感}，高清，真实，商业摄影质感
```

## 参考图再生成公式

```text
参考图风格的{主体}，保留{必须保留的元素}，允许{可以变化的元素}，整体呈现{风格和情绪}，{光线}，{构图}，{材质和色彩}，高清，真实，细节清晰
```

## 负面提示词公式

```text
低清晰度，模糊，结构错误，比例错误，材质错误，颜色偏差，背景杂乱，主体不突出，过曝，欠曝
```

## 写提示词的原则

1. 先写主体，再写风格。
2. 先写必须保留，再写可以变化。
3. 重要内容写具体，不重要内容不写死。
4. 负面提示词只写真实会影响结果的问题。
5. 一条提示词只服务一个明确目标。
