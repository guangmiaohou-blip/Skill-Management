# model-image-generation

## 用途

当用户需要生成模特图、服装展示、电商模特、真人写真、虚拟人、局部模特、饰品展示、发型展示、穿戴展示或生活方式场景图时，使用本 skill。

## 支持类型

| 类型 | 说明 |
| --- | --- |
| 服装模特 | 展示上衣、裤装、裙装、外套、套装等 |
| 电商模特 | 白底、纯色背景、棚拍、平台商品图 |
| 真人写真 | 偏真实摄影、自然场景、生活化表达 |
| 虚拟人 | 不要求真实身份一致的虚拟模特 |
| 局部模特 | 手部、腿部、脚部、背部、肩颈、头发、耳部、颈部等局部展示 |
| 饰品展示 | 耳饰、项链、戒指、手链、发饰、胸针、眼镜、手表、包袋等配饰展示 |
| 发型展示 | 发型、发色、刘海、卷度、长度、发量、发饰搭配、发丝质感展示 |
| 穿戴展示 | 鞋、包、饰品、眼镜、手表、帽子、围巾等穿戴类产品 |
| 生活方式场景 | 模特在真实或拟真场景中展示产品 |

## 输入

```yaml
image: 可选，参考图、产品图、饰品图、发型图、风格图或模特图
user_request: 用户原始需求
model_consistency: 按需求使用
hair_consistency: 按需求使用
accessory_consistency: 按需求使用
language: 中文
```

## 输出

必须输出：

1. 模特图类型。
2. 自动分类结果。
3. 画面目标。
4. 发型与妆容要求。
5. 饰品与配饰要求。
6. 正向提示词。
7. 负面提示词。
8. 推荐参数。
9. 可复用标签。
10. 质量检查清单。
11. 可写入 records 的结构化记录。

## 模特美感与年龄感控制

生成模特图时，Agent 必须先判断用户是否需要：

1. 亚洲感明确的模特。
2. 国际化时尚模特。
3. 年轻甜美感。
4. 成熟冷艳感。
5. 自然真实商业模特。
6. 偶像感近景美妆模特。

如果用户没有明确要求“亚洲女性”或特定族裔，不要默认写死 `East Asian woman`、`phoenix eyes`、`sharp Asian features` 等强族裔特征词。

如果用户要求“更美、更年轻、更像真实模特”，提示词中优先使用：

```text
young fashion model, refined but soft facial features, balanced facial proportions, fresh youthful presence, natural confident expression, modern international editorial beauty, attractive but realistic model, subtle facial asymmetry, real skin texture
```

谨慎使用或避免：

```text
striking, severe, cold, distant, sharp jawline, phoenix eyes, porcelain skin, flawless skin, intense gaze, no fill light
```

这些词会把人物推向成熟、冷硬、亚洲高级脸或 AI 海报感。

## 脸部比例模式

Agent 不要把所有模特图都套用同一种脸部比例。必须根据用户目标选择比例模式。

### 真实商业模特比例

适合电商、服装、真实摄影、成熟时尚模特。

要求：

```text
真实成年人面部比例，自然眼睛大小，正常眼距，协调三庭五眼，真实眼部结构，中庭不过短，脸部不过度幼态
```

避免：

```text
幼态大眼，娃娃脸比例，短中庭，过度放大的瞳孔，美颜滤镜脸，少女漫画感
```

### 偶像近景美妆比例

适合用户明确要求参考图中那种近景偶像感、美妆感、清透少女感、发夹/美甲/手部入镜的脸部特写。

允许并强化：

```text
近景脸部特写，偏幼态但精致的面部比例，眼睛占比略大，清透圆润眼型，短中庭，柔和小脸轮廓，饱满卧蚕，清透眼妆，水润唇妆，轻微无辜感，K-pop idol beauty close-up, glossy beauty editorial
```

但仍需避免：

```text
眼睛畸形，眼距过近，瞳孔过大，眼白过多，五官错位，脸部融化，过度漫画化，假人感
```

当用户提供类似“发夹、粉色美甲、手部贴脸、清透眼妆、近景脸部特写”的参考图时，Agent 应把它识别为“偶像近景美妆比例”，不要误判为普通真实商业模特比例错误。

## 真实感控制

为了避免画面变成 AI 感强的海报，提示词中不要只依赖 `8K`、`ultra realistic`、`flawless`、`porcelain skin`。

真实感优先写：

```text
real camera photograph, natural skin texture, visible pores, subtle skin texture, slight facial asymmetry, realistic fabric texture, natural makeup finish, editorial retouching without plastic skin
```

负面提示词中加入：

```text
AI generated look, plastic skin, waxy skin, over-smoothed skin, uncanny face, overly sharp facial features, harsh artificial beauty, mature severe look
```

## 光线对美感的影响

硬光、无补光和高反差会增强戏剧感，但也会让模特显成熟、显冷、显假。

如果目标是年轻感、美感和真实感，优先使用：

```text
controlled studio light, soft directional key light, subtle fill light, natural catchlights, realistic skin highlights
```

只有用户明确要求强戏剧大片时，才使用：

```text
single hard key light, no fill light, deep Rembrandt shadow, extreme high contrast
```

## 发型控制

如果用户指定发型或上传发型参考，提示词中必须明确：

1. 发型名称或形态，如直发、卷发、盘发、短发、长发、高马尾、低丸子头。
2. 发色和色温，如黑发、棕发、冷棕、金色、挑染。
3. 发丝质感，如顺滑、蓬松、湿发感、自然碎发。
4. 刘海和分缝，如中分、侧分、空气刘海、无刘海。
5. 发型是否需要保持一致。

发型展示任务中，构图应优先保证头发轮廓、发丝层次和发色准确，不要让背景、服装或饰品抢主体。

## 饰品与配饰控制

如果用户指定饰品或上传饰品参考，提示词中必须明确：

1. 饰品类型，如耳环、项链、戒指、手链、发夹、眼镜、手表。
2. 穿戴位置，如左耳、双耳、颈部、手腕、食指、发侧。
3. 材质，如黄金、银色金属、珍珠、宝石、皮革、亚克力。
4. 比例和可见度，如精致小巧、夸张大耳环、主体清晰可见。
5. 反光和高光，如金属高光、宝石折射、珍珠柔光。
6. 是否需要保持款式一致。

饰品展示任务中，构图应优先保证饰品不变形、不丢失、不被头发或手部遮挡。

## 身份一致性

默认按需求使用。如果用户要求同一模特一致，提示词和参数中必须强调：

1. 保持同一面部特征。
2. 保持同一发型和气质。
3. 保持同一身材比例。
4. 尽量保持妆容和肤色。
5. 如有指定饰品，保持饰品款式、位置和材质一致。

如果用户未要求身份一致，不主动锁定具体身份，只保持风格和模特类型一致。

## 质量检查

输出前检查：

1. 手指、四肢、五官是否容易出错。
2. 模特是否符合用户期望的美感、年龄感、族裔感和国际化程度。
3. 人物是否有真实皮肤纹理、自然表情和真实摄影质感。
4. 是否因为提示词过度强调冷艳、强势、硬光，导致人物显成熟或不够美。
5. 发型、发色、刘海和发丝质感是否符合需求。
6. 饰品是否缺失、变形、位置错误、比例失真或被遮挡。
7. 产品结构是否容易变形。
8. 服装版型和材质是否表达清楚。
9. 饰品、发型、服装之间是否风格冲突。
10. 背景是否抢主体。
11. 是否包含负面提示词。
12. 是否能沉淀为可复用案例。
