# 结果校验与重试

本文件只定义所有角色共用的校验顺序与重试框架，不保存任何角色的脸部特征、身体特征、比例值、强制身份标记、资产名称或阈值。所有具体答案必须从当前角色目录读取。

## 公共校验顺序

1. 确认只激活一个角色，并已读取该角色的身份锁定、比例控制、参考图调用、可变外观层和校验规则。
2. 检查身份是否符合当前角色文件，不得引用其他角色的脸、身体、比例或身份标记。
3. 按当前角色的 `比例控制.md` 检查身体比例、头肩关系、颈部、肩宽和肢体长度。
4. 检查当前角色规定的强制身份标记与可选微特征。
5. 检查外观层是否只改变用户允许的项目。
6. 检查产品形状、尺寸、颜色、材质、组件、品牌决策、比例和交互。
7. 检查光源、皮肤高光、阴影、对比度、场景、姿势、构图、镜头、画幅和输出渠道。
8. 全身、七分身或远景任务按当前角色校验文件检查面部可读性、人物画面占比和完整身体边界。
9. 露出身体或颈部时，检查身体真实细节、脸身肤色一致性和颈部结构阴影。
10. 侧脸、回眸、面部回填、头饰、耳机、耳位或下颌任务，调用当前角色的结构检测与局部回填文件；禁止固定指向某个角色目录。
11. 面部细节回填与眼部细节回填分别调用当前角色对应模块，分别校验，禁止合并。
12. 命中当前角色或公共结构文件的一票否决项时直接定向重试，不展示失败样张。

## 公共重试模板

### 身份失败

```text
Regenerate with stronger selected-model-card identity priority. Restore only the face, body frame, proportions, mandatory marks, and appearance values declared by the current model card. Preserve successful pose, product, camera, lighting, and composition. Do not inherit any identity or parameter from another model card.
```

### 全身身份丢失

```text
Use the selected model card's full-body two-stage identity workflow. Keep successful pose, wardrobe, camera, and background. Re-anchor the face from the current role's face master and restore only features, mandatory marks, appearance settings, body ratios, and framing targets declared by that role. Do not preserve genericized facial drift and do not use another role's values.
```

### 身体真实感失败

```text
Keep the selected model card's identity, face, pose, product, background, and composition unchanged. Correct only the body-realism layer using the current role's appearance and validation files. Preserve its body proportions and avoid oily, plastic, over-muscular, or structurally flattened skin.
```

### 头部结构失败

```text
Regenerate using only the selected model card's independent head-structure authority. Rebuild one coherent skull from that role's official structural assets. Apply the shared jaw-chain, ear-position, pose-consistency, and skull-closure method without importing another role's geometry or thresholds. Preserve successful body, product, scene, lighting, and composition.
```

### 面部细节回填失败

```text
Retry only the selected model card's face-detail refill crop. Restore the current role's facial identity, geometry, ordinary facial detail, mandatory marks, skin tone, and texture. Do not load eye-macro assets, do not execute eye-macro validation, and do not redraw the body.
```

### 眼部细节回填失败

```text
Retry only the selected model card's eye-only crop after the face result has passed validation. Preserve the accepted face, eye geometry, gaze, mandatory eye-area marks, body, and composition. Use only the current role's eye assets and eye-macro rules.
```

### 产品失败

```text
Keep the selected model identity unchanged. Restore the supplied product's exact geometry, color, materials, component placement, confirmed branding, and real scale. Correct interaction without inventing accessories, controls, text, or functions.
```

### 预设参考图失败

```text
Regenerate as a controlled preset reference image. Load the selected model card's preset rules, keep every fixed item unchanged, and change only the requested comparison variable.
```
