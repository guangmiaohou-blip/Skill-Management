## 八、关键判断与避坑

1. **气质锁死**：必须用 `striking / intense / confident / powerful` 锁定强势气场，禁用 `cute / kawaii / soft / gentle`。
2. **唇色**：必须显式 `matte crimson-red lips`，禁用 `glossy / shiny / dewy lips`，否则会偏水光唇釉。
3. **肤质**：必须 `matte porcelain-cool skin`，禁用 `dewy / luminous / glowy skin`，否则偏离杂志大片质感。
4. **光线**：必须 `single hard key light + Rembrandt shadow + no fill light`，禁用 `softbox / even lighting / flat lighting`，否则失去戏剧化。
5. **逆光金边**：必须 `strong gold rim backlight`，这是 ELLE 大片的视觉签名。
6. **背景**：必须 `deep charcoal-black seamless`，禁用 `light gray / white / pastel`。
7. **品牌字样**：Nano Banana 对真品牌字样主动拒出，已在负面提示词中显式屏蔽 ELLE / Huawei / ULTIMATE DESIGN + 末尾 `no text, no watermark, no logo` 双保险。
8. **产品屏蔽**：已显式屏蔽 `phone / smartphone / mobile device / handheld device / screen`，避免误生成手机。
9. **杂志留白**：每张末尾都强调 `empty top space for masthead`，保留杂志排版空间。
