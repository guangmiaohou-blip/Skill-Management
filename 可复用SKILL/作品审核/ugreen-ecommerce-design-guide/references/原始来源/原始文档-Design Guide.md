## 3. 字体规范

### 3.1 字体家族

- **指定字体：OPPO Sans 4.0**
- 禁止使用衬线体（宋体、Times、Georgia 等）
- 单张图内字体变化不超过 3 种

### 3.2 字阶系统

| 层级 | 字号 | 字重 | 中文定义 | 适用场景 |
|------|------|------|---------|---------|
| Hero | 40px | SemiBold | 视觉主题 | 首页品牌主视觉大标题 |
| Display | 36px | SemiBold | 强卖点 | 活动模块、品牌 Slogan、核心卖点 |
| Section | 28px | SemiBold | 章节标题 | 详情页分区标题 |
| H1 | 24px | SemiBold | 页面主标题 | 主图核心卖点文字 |
| H2 | 20px | SemiBold | 模块标题 | 主图次要卖点、详情页小标题 |
| H3 | 18px | SemiBold | 卡片标题 | 参数卡片、功能点标题 |
| H4 | 16px | SemiBold | 辅助小标题 | 角标、标签标题 |
| Body_XL | 16px | Regular | 核心叙事 | 核心功能说明、重要段落 |
| Body_L | 16px | Regular | 普通正文 | 普通产品描述 |
| Body_M | 14px | Regular | 次级信息 | 参数说明、规格列表 |
| Body_Bold | 14px | SemiBold | 强调正文 | 正文中的强调内容 |
| Caption | 12px | Regular | 注释 | 脚注、免责声明、图片说明 |
| Label | 12px | SemiBold | 标签 | Tag、Chip、Badge、角标 |

### 3.3 信息层次原则

每张主图信息层次**不超过 3 级**：

```
第1层：产品主体（视觉锚点）
第2层：核心卖点（Hero / Display 字号）
第3层：信任信号（认证、参数、口号，Caption / Label 字号）
```

## 4. 设计原则

### 4.1 占比原则

> 产品主体须占据画面面积的 **60%–70%**

- 产品角度需展现最核心的功能特征：
  - 充电器 → 优先展示接口面
  - 耳机 → 优先展示腔体轮廓
- 用户第一眼必须看到"**这是什么**"，0.3 秒内建立品类认知

### 4.2 品牌识别原则

> 用户需在 **0.3 秒** 内识别"这是绿联"

- 绿联 Logo 必须出现在视觉显著位置（主图左上角）
- 品牌绿 `#007934` 作为主色调或强调色出现在画面中
- 整体视觉语言（材质、光影、构图）须与绿联品牌调性一致

### 4.3 视觉层级原则

> 用户在 **0.5 秒** 内无法处理超过 3 个信息点

- 每个模块信息层级严格控制在 3 层以内
- 模块之间保持足够的视觉呼吸空间
- 禁止在主图中堆砌超过 3 条卖点文字

### 4.4 字体使用原则

| 规则 | 说明 |
|------|------|
| 指定字体 | OPPO Sans 4.0，禁止衬线体 |
| 字体种类 | 同一画面不超过 3 种字体变化 |
| 文字可读性 | 字号最小 12px，与背景对比度符合 WCAG AA 标准 |

### 4.5 产品视觉原则

| 规则 | 说明 |
|------|------|
| 拍摄角度 | **斜俯视图优先**，禁止标准正侧视图 |
| 高光要求 | 高光清晰，细节可见，充分体现材质质感 |
| 光效处理 | 允许光效/材质纹理点缀，但不能喧宾夺主 |
| 世界观统一 | 同一组图光影风格、色调必须统一 |

## 5. 主图布局规范

### 5.1 主图尺寸标准

| 平台 | 主图尺寸 | 最小尺寸 | 格式 |
|------|---------|---------|------|
| 天猫 / 京东 | 800 × 800 px | 800 × 800 px | JPG / PNG，白底 |
| 抖音 | 750 × 750 px | 600 × 600 px | JPG / PNG |
| 官网 banner | 1920 × 600 px | — | JPG / PNG |

### 5.2 800×800 主图标准网格

```
┌─────────────────────────────────────┐
│ 16px                           16px │  ← 安全边距
│ ┌──────────┐                        │
│ │ 绿联 Logo │  ← 左上角固定位置       │
│ └──────────┘                        │
│                                     │
│         产品名称 / 型号              │  ← 小字，Logo 右侧或下方
│                                     │
│   ┌─────────────────────────────┐   │
│   │                             │   │
│   │     核心卖点（Hero 字号）     │   │  ← 左 1/2 区域，大字
│   │                             │   │
│   │  次要卖点 1                  │   │
│   │  次要卖点 2                  │   │  ← Body_M 字号
│   │  次要卖点 3                  │   │
│   │  次要卖点 4                  │   │
│   └─────────────────────────────┘   │
│                                     │
│ 16px                           16px │
└─────────────────────────────────────┘
```

**布局说明：**
- 文字区域：左 1/3 ~ 1/2 宽度
- 产品区域：右 1/2 ~ 2/3 宽度（或居中，文字叠加在产品上方）
- 中轴辅助线：水平 1/2、垂直 1/2 各一条

### 5.3 信息层次模板

```
[产品名] + [型号]              ← Label 12px，低视觉权重
[核心卖点]                     ← Display/H1，最大字号，1~2 行
[次要卖点 × 2~4 条]            ← Body_M 14px，简短短句
[信任信号：认证/奖项/口号]      ← Caption 12px 或图标
```

### 5.4 布局变体

| 布局类型 | 文字区 | 产品区 | 适用场景 |
|---------|-------|-------|---------|
| 左文右图 | 左 40% | 右 60% | 有多行卖点的标准主图 |
| 居中对称 | 上下各 20% | 中间 60% | 单品凸显、高端氛围感 |
| 全图产品 | 叠加在产品上 | 100% | CGI 渲染主图、视觉冲击型 |

## 6. 产品拍摄规范

> 以下规范适用于实拍和 AI 生成出图，AI 提示词须严格对应各条目。

### 6.1 角度规范

| 角度类型 | 说明 | 优先级 |
|---------|------|-------|
| 斜俯视 3/4 角（推荐） | 水平面 15°~30° 侧转，略微俯视 | ★★★ 最高 |
| 正面微侧 | 近乎正视，轻微 5°~10° 侧转 | ★★ |
| 低角度仰视 | 突出产品质量感与体积感 | ★ |
| 标准正侧视 | **禁用**——过于平淡，信息量低 | ✕ 禁止 |

### 6.2 背景规范

| 背景类型 | 技术要求 | 适用风格 |
|---------|---------|---------|
| 纯白白底 | RGB 255·255·255，无渐变 | 电商标准主图（平台首图必选） |
| 极浅灰 | RGB ≥ 240·240·240 | 商业精修、高端品质感 |
| 渐变背景 | 单色渐变，不超过 2 色 | 氛围主图、详情页 |
| 场景化 | 真实或 CGI 场景 | 生活方式图，不用于平台首图 |

### 6.3 光影规范

| 规则 | 说明 |
|------|------|
| 色温标准 | 5500K 日光标准，禁止冷暖倾向 |
| 阴影处理 | 仅允许极淡的产品底部接触阴影（暗示"放置感"），禁止硬投影 |
| 高光分布 | 均匀分布全表面，无"视觉优先级"的明暗取舍 |
| 背景光 | 独立光源照亮背景，确保纯白无渐变污染 |

### 6.4 材质还原标准

| 材质类型 | 还原要求 |
|---------|---------|
| 金属（铝合金/不锈钢） | 拉丝方向清晰，阳极氧化微粒感，CNC 倒角高光线可见 |
| 塑料（磨砂/亮面） | 区分亮面与磨砂光泽差异，注塑合模线清晰但不突兀 |
| PVC / 编织线缆 | 每股纤维可追踪，编织纹路规律清晰 |
| Logo & 丝印 | 字体边缘锐利，油墨质感可辨（Pantone 级色彩准确度） |
| 接口内部 | USB-C / HDMI 等接口内金属触点、塑料隔板必须清晰呈现 |

### 6.5 AI 出图提示词对照表

基于本规范，AI 出图时各字段的提示词对应关系：

| 设计规范字段 | AI 提示词写法 |
|------------|-------------|
| 斜俯视 3/4 角 | `three-quarter view, slight overhead angle, 20° tilt` |
| 纯白背景 | `pure white background, RGB 255 255 255, no shadows, no gradients` |
| 5500K 色温 | `daylight 5500K white balance, neutral color, no warm/cool tint` |
| 材质可辨 | `surface texture visible, material details accurate, micro texture` |
| 零变形 | `exact product proportions, no distortion, accurate logo placement` |
| 商业目录级 | `commercial catalog quality, industrial precision, pixel-sharp` |

## 附录：快速检查清单

在提交任何主图前，逐项确认：

**品牌符号**
- [ ] 绿联 Logo 清晰出现在左上角
- [ ] Logo 未变形、未遮挡、未与背景同色

**色彩**
- [ ] 主色为品牌绿系（`#007934` / `#009632`）或白底
- [ ] 背景白度 ≥ RGB 240（纯白底图）

**文字**
- [ ] 使用 OPPO Sans 4.0
- [ ] 字体种类 ≤ 3 种
- [ ] 信息层次 ≤ 3 层

**产品**
- [ ] 产品主体占比 60%–70%
- [ ] 角度为斜俯视（非正侧视）
- [ ] Logo / 接口 / 材质细节清晰可辨

**技术**
- [ ] 尺寸 ≥ 800 × 800 px
- [ ] 文件大小 > 100KB（防空图）
- [ ] 背景四角亮度 > 230/255
