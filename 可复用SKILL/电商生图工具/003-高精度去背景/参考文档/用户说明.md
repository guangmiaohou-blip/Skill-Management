# ✂️ 高精度去背景 — 用户说明

> 此文件仅供用户阅读，AI 模型不会加载此文档。

## 这个 Skill 是干什么的？

用 **BiRefNet 模型** 把图片的背景去掉，生成**透明 PNG**。专门给电商产品图、人像照片抠图用。

## 和普通去背景工具有什么不同？

| 对比项 | 普通工具 | BiRefNet（本 Skill） |
|--------|---------|---------------------|
| 精度 | 头发丝容易抠坏 | **高精度**，发丝边缘清晰 |
| 复杂背景 | 纯色背景效果好 | 复杂背景也 hold 住 |
| 产品图 | 边缘可能残留 | 干净利落 |

## 典型用法

产品摄影 → 去背景 → 贴到任意背景上

电商常用场景：
- 白底图制作（去掉原有背景，换纯白底）
- 场景图合成（抠出产品→贴到生活场景中）
- 多平台适配（一张抠图→不同平台不同背景）

## 需要什么？

| 项目 | 说明 |
|------|------|
| CLI 工具 | 需要安装 inference.sh CLI |
| 安装命令 | `curl -fsSL https://cli.inference.sh | sh && infsh login` |
| 计费 | inference.sh 平台按次计费 |

## 命令示例

```bash
infsh app run infsh/birefnet --input '{"image_url": "https://your-photo.jpg"}'
```
