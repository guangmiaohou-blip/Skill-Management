## 功能概述

用 BiRefNet 模型高精度去除图片背景，生成透明 PNG。适用于产品图、人像、电商图抠图。

## 前置条件

## 使用方式

```bash
infsh app run infsh/birefnet --input '{"image_url":"https://..."}'
```
