## 功能概述

AI驱动的时尚模特姿态迁移工具。用参考姿态图生成模特/产品图像的姿态变化，同时保留服装和背景。需 FAL_KEY。

## 前置条件

FAL_KEY (FAL.ai), Python 3.8+, fal-client>=0.8.0

## 使用方式

```bash
python3 scripts/generate.py --original model.jpg --pose ref.jpg
```
