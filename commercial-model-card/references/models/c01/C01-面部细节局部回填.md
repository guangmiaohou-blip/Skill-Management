# C01 面部细节局部回填

这是独立的完整面部细节回填模块。用于全身图、远景图或低像素面部中 C01 身份、面部几何、肤质、雀斑、鼻唇和强制身份标记丢失时，通过“面部裁切 → 放大 → 面部回填 → 缩回 → 柔边合成”恢复整脸。

## 触发与边界

- 用户要求“面部回填”“面部细节回填”“整脸回填”或明确对完整面部执行裁切—放大—回填—合成时，触发本模块。
- 本模块不触发 `C01-眼部微距局部回填.md`，不加载眼部微距母版，不生成虹膜、角膜、泪膜、睑缘或单根睫毛微结构。
- 眼睛在本模块中只作为身份几何：保持眼型、眼距、眼角坐标、睁眼程度、双眼皮主路径、视线和普通可见效果。
- 冻结身体、颈部、姿势、服装、产品、构图和背景。需要校正头骨、耳位、下颌或脸身肤色时，另行读取 `C01-局部回填与头部结构校验.md`。

## 执行

1. 裁切完整面部及必要的额头、太阳穴和下颌过渡，保留足够配准锚点；不要把肩膀、大片头发或身体纳入生成区域。
2. 等比例放大裁片；放大只提供工作像素，不改变五官坐标、脸宽、头骨或表情。
3. 使用 C01 身份母版恢复脸型、三庭五眼、五官落点、发际线、鼻唇、右眼眼皮痣、雀斑、肤色、肤质和普通面部清晰度。
4. 明确禁止调用或引用眼部微距资产；不得为了增加眼部微结构而改变整脸。
5. 缩回原尺寸并匹配原图肤色、综合色温、明度、对比度和锐度。
6. 只在完整面部内部使用柔边合成；耳朵、鬓角、头发边缘、下颌外轮廓和颈部默认保留原图，除非它们属于经确认的结构修复目标。

## 校验与重试

- 检查 C01 身份、三庭五眼、五官坐标、脸型、右眼眼皮痣、雀斑、肤质、肤色和表情。
- 检查合成边界是否无重影、错位、色阶断层、锐度断层或混合脸。
- 不检查虹膜纤维、角膜、泪膜、睑缘或单根睫毛微结构。
- 失败时只重做面部裁片，不触发眼部模块，不重绘身体。
- 用户同时要求眼部回填时，先完成并验收本模块，再把最终图交给眼部模块独立处理。
