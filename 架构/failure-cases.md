# 失败案例与避坑规则

## 人体类

常见问题：

1. 手指数量错误。
2. 四肢扭曲。
3. 五官不自然。
4. 姿势和服装结构冲突。

负面提示词建议：

```text
手指错误，手部畸形，四肢扭曲，五官变形，人体比例错误，多余肢体
```

## 产品类

常见问题：

1. 产品结构变形。
2. logo 或文字乱码。
3. 材质不准确。
4. 产品边缘模糊。

负面提示词建议：

```text
产品变形，结构错误，文字乱码，logo 变形，材质错误，边缘模糊
```

## 风格类

常见问题：

1. 过度艺术化。
2. 商业质感不足。
3. 背景抢主体。
4. 色彩偏离参考图。

负面提示词建议：

```text
低质感，过度艺术化，背景杂乱，主体不突出，颜色偏差，过曝，欠曝
```
