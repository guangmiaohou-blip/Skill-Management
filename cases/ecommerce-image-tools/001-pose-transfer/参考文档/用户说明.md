# 💃 姿态迁移工具 — 用户说明

> 此文件仅供用户阅读，AI 模型不会加载此文档。

## 这个 Skill 是干什么的？

给电商服装模特的**照片换姿势**。你有一张模特穿着某件衣服的照片，再给一张参考姿势图，AI 会把模特"掰"成参考图的姿势，同时**衣服和背景保持不变**。

## 典型用法

产品摄影 → 去背景 → 姿态迁移 → 出多张不同姿势的模特图

比如你拍了一件 T 恤的平铺图：
1. 先 `ai-product-photography` 生成棚拍图
2. 再 `background-removal` 抠成透明底
3. 最后 `pose-transfer` 生成站姿 / 坐姿 / 走姿等多套姿势

## 需要什么？

| 项目 | 说明 |
|------|------|
| API Key | 需要 `FAL_KEY`（FAL.ai 平台的 API 密钥）|
| Python | 需要 Python 3.8+ |
| 依赖 | `pip install fal-client>=0.8.0` |

## 命令示例

```bash
python3 scripts/generate.py \
  --original "model.jpg" \
  --pose "reference_pose.jpg" \
  --output-dir "./output"
```
