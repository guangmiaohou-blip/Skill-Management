# AI产品摄影 — 用 FLUX/Imagen 3/Grok/Seedream 等模型生成专业AI产品摄影图

## 功能

用 FLUX/Imagen 3/Grok/Seedream 等模型生成专业AI产品摄影图。支持棚拍灯光、生活场景、包装展示、Mockup。

## 用法

```bash
infsh app run falai/flux-dev --input '{"prompt":"...","size":"2K"}'
```

## 依赖

inference.sh CLI (curl -fsSL https://cli.inference.sh | sh && infsh login)
