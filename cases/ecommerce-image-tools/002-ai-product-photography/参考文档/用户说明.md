# 📸 AI 产品摄影 — 用户说明

> 此文件仅供用户阅读，AI 模型不会加载此文档。

## 这个 Skill 是干什么的？

用 AI **生成专业级的产品摄影图**，不需要实拍，也不需要摄影棚。

支持多个顶尖生图模型：

| 模型 | 特点 |
|------|------|
| **FLUX** (falai/flux-dev) | 高质量真实感，适合电商主图 |
| **Imagen 3** | Google 出品，光影自然 |
| **Grok** (xai/grok-2a) | xAI 模型，风格多样 |
| **Seedream** (bytedance/seedream-4-5) | 字节跳动，东方审美适配 |

## 能力范围

- 棚拍灯光效果（柔光/硬光/侧光）
- 生活场景图（咖啡厅/客厅/户外）
- 产品包装展示
- 电商主图 / Amazon Listing 图
- 产品 Mockup

## 需要什么？

| 项目 | 说明 |
|------|------|
| CLI 工具 | 需要安装 inference.sh CLI |
| 安装命令 | `curl -fsSL https://cli.inference.sh | sh && infsh login` |
| 计费 | inference.sh 平台按次计费 |

## 命令示例

```bash
# 棚拍风格产品图
infsh app run falai/flux-dev --input '{
  "prompt": "Professional product photo of wireless earbuds on white surface, soft studio lighting, commercial photography, high detail",
  "size": "2K"
}'
```
