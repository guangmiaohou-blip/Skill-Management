# UNNY 韩系极简美妆测试案例

用途：作为 `unny-korean-minimal-beauty` 风格 Skill 的训练来源和审核样例。

## 1. 目的

验证无产品变体的人像生成在高、中、低相似度下是否能稳定保持 UNNY 韩系极简美妆视觉方向。

## 2. 适用范围

适用于模特图、人像图、美妆图和参考图再生成。默认去除参考图中的产品、文字、图形、Logo 和水印。

## 3. 执行规则

该案例基于 UNNY 韩系极简美妆 DNA，去除产品后生成高/中/低相似度各 1 张人像。

## 4. 正向规则

核心特征：

1. 纯白无缝棚拍背景。
2. 大地色低饱和配色。
3. 韩系极简美妆编辑片风格。
4. 柔和前左侧漫射光。
5. 高相似保留发型、妆容、姿态、光线和配色。
6. 中相似保留核心 DNA，放宽姿态和表情。
7. 低相似只保留氛围、纯白棚拍、大地色和柔和光线。

## 5. 负面规则

- 不保留参考图中的产品本体、品牌、文字、Logo、图形或水印。
- 避免塑料皮肤、过度成熟、过度夸张的偶像脸和不自然的五官比例。

## 6. 参数与示例

```yaml
size: "1024x1365"
aspect_ratio: "3:4"
quality: high
background: opaque
similarity_levels:
  - high
  - mid
  - low
product: none
```

## 7. 质量检查

1. 纯白棚拍背景稳定，画面干净。
2. 人物样貌、妆容和发型符合所选相似度档位。
3. 皮肤、眼睛和五官具有真实摄影质感。
4. 参考图中的产品、文字、图形、Logo 和水印未被保留。

## 8. 限制与待确认项

- 平台和模型未固定，需在具体任务中确认。
- 推荐参数中的 `style_strength` 仍需按平台确认。
- 本记录尚未审核为正式知识。
