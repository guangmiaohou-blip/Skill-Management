# reference-image-regeneration

## 用途

当用户上传参考图，并希望重新生成相似图、保留人物样貌、保留风格、复刻视觉方向或基于参考图再创作时，使用本 skill。

## 输入

```yaml
image: 用户上传参考图
user_request: 用户原始需求
similarity_level: 高相似 | 中相似 | 低相似 | 待确认
language: 中文
```

如果用户没有指定相似度，必须先询问相似度。不要默认使用任何相似度档位。

询问话术：

```text
这张参考图再生成时，你希望相似到什么程度？

A. 高相似：样貌、脸部比例、五官关系、气质、发型妆容尽量保持；动作姿势仅在你要求时保持
B. 中相似：保留样貌方向、气质、发型妆容和风格，允许动作、手势、背景、构图变化
C. 低相似：只保留风格方向和情绪，允许重新创作样貌、动作和场景
```

## 输出

必须输出：

1. 图片类型。
2. 自动分类结果。
3. 相似度档位。
4. 样貌保留项。
5. 动作与构图是否保留。
6. 文字、logo、图形去除项。
7. 正向提示词。
8. 负面提示词。
9. 推荐参数。
10. 可复用标签。
11. 生成执行建议。
12. 可写入 records 的结构化记录。

## 相似度定义

参考图再生成的相似度优先指“人物样貌与视觉身份相似”，不是默认复刻动作。

人物图、模特图、人像图中，相似度优先保留：

1. 脸部比例。
2. 五官关系。
3. 眼型、鼻型、唇形、脸型。
4. 年龄感、气质和美感方向。
5. 发型、发色、妆容和肤色。
6. 整体风格和摄影质感。

默认不强制保留：

1. 具体动作。
2. 手势。
3. 站姿或坐姿。
4. 机位。
5. 构图。
6. 道具。

只有用户明确要求“动作也保持一致”“同姿势”“同构图”时，才把动作姿势和构图加入保留项。

## 文字与图形去除规则

参考图中出现的以下内容默认全部去除，不保留：

```text
文字，logo，品牌字样，水印，图形符号，UI 图标，排版文字，装饰性图案，无关标识，屏幕内容，海报字样
```

这些内容必须写入负面提示词或避免项。只有用户明确要求保留时，才能保留，并标记为 `用户明确要求保留`。

## 产品去除规则

参考图中的产品默认全部去除，不保留。产品包括但不限于：

```text
手机，耳机，充电宝，平板，电脑，包装盒，手持设备，穿戴设备，品牌物料，道具商品
```

如果人物动作与产品有关，Agent 应保留人物动作和肢体关系，但移除产品：

```text
手持产品 -> 保留手部姿势和展示动作，去除产品
佩戴产品 -> 保留身体姿态和展示角度，去除产品
靠近产品 -> 保留人物视线和互动关系，去除产品
```

只有用户明确要求保留产品时，才可保留产品，并在输出中标注 `用户明确要求保留产品`。

## 相似度控制

### 高相似

保留：

1. 人物样貌、脸部比例和五官关系。
2. 年龄感、气质、美感方向。
3. 发型、发色、妆容和肤色。
4. 主体身份或产品外观。
5. 核心风格、光影氛围、材质和颜色。
6. 用户明确要求保留的构图、动作或机位。

### 中相似

保留：

1. 样貌方向和脸部比例方向。
2. 气质、年龄感和美感方向。
3. 发型妆容方向。
4. 主体类型和核心风格。
5. 光线氛围、关键材质或颜色。

允许变化：

1. 背景细节。
2. 姿势或动作。
3. 手势。
4. 构图和机位。
5. 道具。
6. 次要颜色。

### 低相似

保留：

1. 风格方向。
2. 情绪。
3. 色彩倾向。
4. 商业用途。

允许重新创作样貌、动作、构图和场景。

## 执行步骤

1. 先使用 `reference-image-to-prompt` 的分析维度拆解参考图。
2. 判断参考图是否为人物图、模特图或人像图。
3. 根据相似度档位筛选样貌保留项、风格保留项和允许变化项。
4. 将动作、姿势、构图和机位标记为“仅用户明确要求时保留”。
5. 将参考图中的文字、logo、图形符号、水印和无关标识放入必须避免项。
6. 将参考图中的产品放入必须去除项；如人物正在手持或展示产品，保留人物动作和手部姿态。
7. 生成正向提示词和负面提示词。
8. 给出推荐参数。
9. 如平台支持参考图权重，给出参考图权重建议。
10. 输出可复用记录。

## 参数建议

```yaml
high_similarity:
  image_reference_weight: 高
  face_identity_weight: 高
  pose_reference_weight: 仅用户要求时使用
  style_strength: 中
  variation_strength: 低
medium_similarity:
  image_reference_weight: 中
  face_identity_weight: 中高
  pose_reference_weight: 低或关闭
  style_strength: 中
  variation_strength: 中
low_similarity:
  image_reference_weight: 低
  face_identity_weight: 低
  pose_reference_weight: 关闭
  style_strength: 中高
  variation_strength: 高
```

## 质量检查

生成前检查：

1. 是否明确保留主体。
2. 对人物图，是否明确相似度主要体现在样貌、脸部比例、五官关系、气质、发型妆容上。
3. 是否把动作姿势、构图和机位误当成默认保留项。
4. 是否明确允许变化的部分。
5. 是否去除参考图文字、logo、图形符号、水印、UI 和无关标识。
6. 是否去除参考图产品，同时保留必要的人物动作和手部关系。
7. 是否给出负面提示词。
8. 是否记录平台、模型、日期、操作人和版本。
