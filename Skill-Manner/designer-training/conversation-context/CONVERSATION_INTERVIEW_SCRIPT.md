# 对话训练追问脚本

## 使用方式

当 Agent 从对话上下文中无法判断训练目标时，最多问 3 个问题。

## 默认追问

```text
我可以把这段对话整理成训练材料。为了避免误写规则，我只确认 3 点：

1. 这段对话主要要训练哪个能力？
   A. 参考图逆向提示词
   B. 参考图再生成
   C. 模特图生成
   D. 新 Skill
   E. 不确定

2. 这段经验更像什么？
   A. 好案例
   B. 失败案例
   C. 通用规则
   D. 流程经验

3. 你希望我怎么处理？
   A. 更新已有 Skill 候选
   B. 创建新 Skill 草案
   C. 只生成待审核知识库候选
```

## 确认话术

```text
我已经从对话里整理出这条训练经验：

设计师明确提到：【xxx】
我根据上下文推断：【xxx】
仍需确认：【xxx】

我建议将它作为【更新已有 Skill / 新 Skill 草案 / 知识库候选】，并放入待审核队列。
这个总结是否准确？
```
