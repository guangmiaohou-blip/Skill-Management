# 失败案例与避坑规则

## 人体类

常见问题：

1. 手指数量错误。
2. 四肢扭曲。
3. 五官不自然。
4. 姿势和服装结构冲突。

负面提示词建议：

```text
手指错误，手部畸形，四肢扭曲，五官变形，人体比例错误，多余肢体
```

## 产品类

常见问题：

1. 产品结构变形。
2. logo 或文字乱码。
3. 材质不准确。
4. 产品边缘模糊。

负面提示词建议：

```text
产品变形，结构错误，文字乱码，logo 变形，材质错误，边缘模糊
```

## 风格类

常见问题：

1. 过度艺术化。
2. 商业质感不足。
3. 背景抢主体。
4. 色彩偏离参考图。

负面提示词建议：

```text
低质感，过度艺术化，背景杂乱，主体不突出，颜色偏差，过曝，欠曝
```

## 模特美感与真实感类

常见问题：

1. 亚洲特征被过度强化，导致模特感变窄。
2. 过度强调冷艳、强势、距离感，导致人物显成熟、不够年轻。
3. 使用 `flawless`、`porcelain skin`、`8K`、`ultra realistic` 等词后，皮肤变成塑料感或蜡像感。
4. 单灯硬光、无补光、高反差过强，导致脸部结构变硬，真实感下降。
5. 五官线条过锋利，表情僵硬，缺少自然生命力。

避坑规则：

```text
如果用户没有明确要求特定族裔，不要默认写死 East Asian woman、phoenix eyes、sharp Asian features。
如果目标是年轻感和美感，不要把 striking、cold、distant、severe、intense gaze 作为主导气质。
如果目标是真实感，不要只依赖 flawless、porcelain skin、8K、ultra realistic，应加入 natural skin texture、visible pores、subtle facial asymmetry、real camera photograph。
```

负面提示词建议：

```text
AI生成感，塑料皮肤，蜡像皮肤，过度磨皮，毛孔完全消失，脸部过度锋利，年龄感过熟，表情僵硬，冷硬距离感过强，亚洲特征过度强化，不自然五官，假人感
```

## 脸部比例模式误判类

常见问题：

1. 用户需要偶像近景美妆比例时，Agent 错误套用真实商业模特比例，导致画面不够像参考图。
2. 用户需要真实商业模特比例时，Agent 错误套用清新少女或偶像美妆比例，导致眼睛过大、中庭过短、脸部幼态。
3. Agent 没有先判断任务是“近景脸部特写”还是“服装/产品商业模特图”。

判断规则：

```text
如果参考图包含近景脸部特写、发夹、美甲、手部靠近脸部、清透眼妆、水润唇妆，可以识别为偶像近景美妆比例。
如果任务是服装、电商产品、真实模特展示，应优先使用真实商业模特比例。
```

偶像近景美妆比例允许：

```text
眼睛占比略大，短中庭，清透圆润眼型，饱满卧蚕，小脸轮廓，清透少女感
```

但仍需避免：

```text
眼睛畸形，眼距过近，瞳孔过大，眼白过多，五官错位，过度漫画化，假人感
```

## 相似度误解类

常见问题：

1. 把参考图相似度错误理解为动作、姿势、构图完全一致。
2. 忽略人物样貌、脸部比例、五官关系、气质、发型妆容这些真正需要相似的内容。
3. 把参考图中的文字、logo、图形符号、水印、UI 或排版元素误当成保留项。

正确规则：

```text
人物图的相似度优先体现在样貌、脸部比例、五官关系、气质、发型妆容和整体美感方向上。
动作、手势、构图和机位只在用户明确要求时保留。
参考图中的文字、logo、图形符号、水印、UI、排版文字和无关标识默认全部去除。
```

负面提示词建议：

```text
文字，logo，品牌字样，水印，图形符号，UI图标，排版文字，装饰性图案，无关标识，屏幕内容，海报字样，复制参考图文字，复制参考图图案
```

## 参考图产品误保留类

常见问题：

1. 参考图中的手机、耳机、充电宝、包装盒、平板等产品被误当成保留项。
2. 去除产品时，把人物手部动作、视线方向或展示姿态也一起破坏。
3. 生成结果出现参考图品牌、产品形状、屏幕内容或包装图形。

正确规则：

```text
参考图中的产品默认全部去除。
如果人物正在手持、佩戴、展示或靠近产品，应保留人物动作、手部位置、身体姿态和互动关系，但移除产品本体。
只有用户明确要求保留产品时，才把产品加入保留项。
```

负面提示词建议：

```text
手机，耳机，充电宝，平板，电脑，包装盒，手持设备，穿戴设备，产品本体，品牌物料，产品logo，屏幕内容，包装图形，商业标识
```
